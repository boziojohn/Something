# AudioYawn 🎵

**AudioYawn** is a simple command-line tool written in C for processing WAV audio files.  
It allows you to **inspect**, **reverse**, **convert to mono**, and **crop** WAV files easily.

---

## Features

- **List** the header and contents of a WAV file  
- **Reverse** an audio file (play backward)  
- **Convert to mono** if the audio has multiple channels  
- **Crop** a section of audio between a start and end time
